Welcome!

This is a web app for Hidden Cannon Chess, a chess variant I invented inspired by Xiangqi. Read the rules HERE.

This app is built using Flask and Flask-SocketIO, as well as chessboard.js and python-chess. Check out the source code on GitHub.

Via the seat mechanic, inspired by Variant go server, two people can play each other on the same computer or from different continents.

All sound effects are taken from Lichess.

Happy gaming!
