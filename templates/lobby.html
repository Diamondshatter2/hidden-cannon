Welcome!

This is a web app for Hidden Cannon Chess, a chess variant I invented inspired by Xiangqi. Read the rules HERE.

This app is built using Flask and Flask-SocketIO, as well as python-chess and chessboard.js. The sound effects are taken from Lichess.

Support for mobile devices is not yet implemented, so please switch to a computer if you're on mobile.

Check out the source code on GitHub if you're, like, a nerd or something.

Happy gaming!
